Easily add announcements in the form of banners and popups to your WordPress site.

== Description ==

Upgrade your site with customizable announcements in 3 configurable areas (more to come in future updates) and popups.

= Features =

* Add banners anywhere on your site
* Configurable colors, sizes, and alignments for banners
* Choose which pages to show announcement on or which pages to hide on
* Optionally dismissible banners that retain memory on being dismissed by user
* Set start and expiration dates and times of announcements
* Compatible with any theme, and if using a common theme will automatically configure self correctly

= Setup =

Once you install and active please be sure to visit the settings page to configure the plugin before adding your announcements or popups. Documentation is coming soon on this process.
